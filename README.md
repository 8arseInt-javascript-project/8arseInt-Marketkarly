**Note**: 해당 프리셋은 node_modules를 내장하고 있지 않습니다. 다운받아 그대로 사용할 경우 `npm install` 을 하신 후 사용해 주세요.

<br/>

### 백엔드 서버 실행

```bash
npm run serve:backend
```

<br/>

### 프론트 서버 실행

```bash
npm run serve:frontend
```

<br/>

### tailwind 실행

```bash
npm run tailwind
```

<br/>

### 동시 실행

```bash
npm run start
```
