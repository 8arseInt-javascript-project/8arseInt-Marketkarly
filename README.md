### 서버 실행

#### 패키지 설치

```bash
npm install
```

#### 백엔드 서버 실행

```bash
npm run serve:backend
```

#### 프론트 서버 실행

```bash
npm run serve:frontend
```

#### 동시 실행

```bash
npm run start
```
